📊 Scatter Plot คืออะไร?

กราฟกระจาย (Scatter Plot) เป็นกราฟที่แสดงความสัมพันธ์ระหว่างตัวแปร 2 ตัว โดยใช้จุดในระบบพิกัด

  • ประโยชน์: ดูแนวโน้มและความสัมพันธ์ของข้อมูล
  • ใช้เมื่อ: ต้องการศึกษาว่าตัวแปรหนึ่งมีผลต่ออีกตัวแปรหนึ่งหรือไม่
  • ตัวอย่าง: ความสัมพันธ์ระหว่างเวลาศึกษากับคะแนนสอบ
  • ข้อดี: เห็นภาพรวมข้อมูลได้ชัดเจน สามารถหาค่าผิดปกติได้

🔍 ค่าสหสัมพันธ์คืออะไร?

ค่าสหสัมพันธ์ (Correlation Coefficient) หรือ r เป็นค่าที่บอกความแข็งแกร่งและทิศทางของความสัมพันธ์เชิงเส้นตรงระหว่างตัวแปร 2 ตัว

  • ช่วงค่า: -1 ≤ r ≤ 1
  • r = 1: ความสัมพันธ์เชิงบวกสมบูรณ์
  • r = 0: ไม่มีความสัมพันธ์เชิงเส้นตรง
  • r = -1: ความสัมพันธ์เชิงลบสมบูรณ์

📊 การแปลความหมาย

0.7 ≤ |r| ≤ 1.0 ความสัมพันธ์แข็งแกร่ง
0.3 ≤ |r| < 0.7 ความสัมพันธ์ปานกลาง
0.0 < |r| < 0.3 ความสัมพันธ์อ่อน
r = 0 ไม่มีความสัมพันธ์

🧮 สูตรการคำนวณ (Pearson's r)

r = n∑XY - ∑X∑Y √[(n∑X² - (∑X)²)(n∑Y² - (∑Y)²)]

โดยที่:

  • n = จำนวนจุดข้อมูล
  • X, Y = ค่าของตัวแปร X และ Y
  • ∑XY = ผลรวมของ X×Y
  • ∑X, ∑Y = ผลรวมของ X และ Y
  • ∑X², ∑Y² = ผลรวมของ X² และ Y²
